Redundant Multi‑ISP Network Infrastructure with Advanced Monitoring

Designed and implemented a resilient network infrastructure using five ISPs for redundancy, Windows Server DHCP, VLAN segmentation, traffic shaping, and PRTG monitoring with automated notifications.

Topology Diagram

Overview

I architected a robust enterprise network infrastructure to ensure high availability and optimized performance across multiple offices and departments. The design leveraged five Internet Service Providers (ISPs) configured for redundancy, guaranteeing continuous connectivity even during outages. By combining Windows Server DHCP, VLAN segmentation, traffic shaping, and PRTG monitoring, the solution delivered secure, efficient, and transparent network operations.

Goals

  • Provide uninterrupted internet connectivity through multi‑ISP redundancy.
  • Centralize IP address management with Windows Server DHCP.
  • Segment traffic by department using VLANs for security and performance.
  • Shape and prioritize traffic for critical applications and VIP subnets.
  • Monitor infrastructure health with PRTG and enable proactive notifications.
